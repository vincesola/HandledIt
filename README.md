# HandledIt Brand Assets

Approved direction:
- Rounded indigo/purple square
- White `H`
- Subtle checkmark integrated into the H
- Calm, non-white UI background

## Use these files

### App icon
Copy `AppIcon.appiconset` into:

`HandledIt/Assets.xcassets/AppIcon.appiconset`

### Brand
Use:
- `Brand/handledit-brand-mark-1024.png`
- `Brand/handledit-brand-mark-512.png`
- `Brand/handledit-horizontal-lockup.png`
- `Brand/handledit-brand-mark.svg`

### Splash
Use:
- `Splash/handledit-splash-concept.png` as reference
- `Splash/splash-animation-spec.json` as timing guidance
